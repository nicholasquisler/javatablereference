Java Reference Sheet

String Methods

Command Description
length() Returns total number of characters.
concat() Appends one string to the end of another.
equals() A boolean to test for equality between two case-matching strings.
equalsIgnoreCase() A boolean to test for equality between two non-case-matching strings.
indexOf() Returns first occurance index of a character or substring.
charAt() Returns a string's characters at a specified index.
toUpperCase() Converts a string to upper case.
toLowerCase() Converts a string to lower case.
